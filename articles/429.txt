Ряд Штурма (система Штурма) для вещественного многочлена — последовательность многочленов, позволяющая эффективно определять количество корней многочлена на промежутке и приближённо вычислять их с помощью теоремы Штурма➤. Ряд и теорема названы именем французского математика Жака Штурма, определившего ряд и его свойства, а также разработавшего конструктивный способ построения такого ряда в 1829 году. Рассмотрим многочлен f ( x ) {\displaystyle f(x)} с вещественными коэффициентами. Конечная упорядоченная последовательность отличных от нуля многочленов с вещественными коэффициентами называется рядом Штурма для многочлена f ( x ) {\displaystyle f(x)} , если выполнены следующие условия: Значением ряда Штурма в точке c {\displaystyle c} называется количество смен знака в последовательности f 0 ( c ) , f 1 ( c ) , … , f s ( c ) {\displaystyle f_{0}(c),f_{1}(c),\dots ,f_{s}(c)} после исключения нулей. Иногда ряд Штурма также определяют как построенный определённым образом➤ ряд Штурма.
